NutriPlan – Smart Meal Planner

Description
NutriPlan is a web-based meal planner that helps users track calories, set diet preferences, and receive smart food recommendations.

Features
* User login & onboarding
* Diet selection (classic, low-carb, vegetarian, vegan)
* Calorie tracking
* Smart meal recommendations
* Favorites system
* Recipe suggestions
* Weekly reminder setup

Technologies Used
* HTML, CSS, JavaScript
* Node.js + Express
* JSON Server (db.json)

How to Run
1. Install dependencies
npm install

2. Run the development server:
npm run dev

3. Open in browser
http://localhost:5000

Project Structure
* public/ → frontend files
* server.js → backend server
* db.json → database
* README.md → documentation
